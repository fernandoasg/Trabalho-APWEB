**PostgreSQL**

Procura o arquivo php.ini no diretorio do seu PHP e descomente a seguinte linha:
- extension=pdo_pgsql

No arquivo .env do seu projeto laravel mude:
- DB_CONNECTION=pgsql

Em config/database.php deixe:
- 'default' => env('DB_CONNECTION', 'pgsql')
